AI Academic Advisor System

An intelligent web-based Academic Advisor platform developed using HTML, CSS, JavaScript, Java Servlets, JDBC, and MySQL. The system helps students manage academic activities through an interactive dashboard with features like course enrollment, appointment scheduling, analytics visualization, and AI-powered guidance. It provides a user-friendly interface for students to access educational resources, monitor progress, and improve decision-making in academics.

Key Features:

AI-powered student assistance and recommendations
Interactive dashboard with analytics charts
Course enrollment and management system
Appointment booking functionality
Student performance tracking and reports
Secure database integration using MySQL
Responsive and user-friendly interface

Technology Stack:
Frontend: HTML, CSS, JavaScript
Backend: Java Servlet, JDBC
Database: MySQL
Server: Apache Tomcat

Purpose:
The project aims to simplify academic management by combining traditional student services with intelligent AI-based assistance to improve learning efficiency and user experience.
